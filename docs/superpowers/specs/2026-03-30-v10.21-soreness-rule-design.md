# KneeHab v10.21 — Soreness Rule

**Date:** 2026-03-30
**Version bump:** Minor → v10.21
**Base version:** v10.2

---

## Problem

The program has no defined protocol for sore Load days. Without guidance, the user either pushes through dynamic loading (risks aggravation) or skips entirely (loses collagen synthesis window and tissue stimulus). A structured soreness modifier preserves the loading day while removing the dynamic/eccentric demand that worsens soreness.

---

## Changes

### 1. Load Day briefing — Soreness Rule

Add after the Asymmetry Rule in the BLOCKS 'A' note field:

> **Soreness Rule:** If muscle soreness >2/10 on a scheduled Load day, replace dynamic exercises with their isometric modifier (listed in each exercise). If soreness is articular (behind the kneecap or inside the joint), switch to Recovery day instead.

---

### 2. Isometric modifiers — 5 exercise descriptions updated

Each dynamic exercise in the Knee block gets one added sentence in its `d` field, both in BLOCKS JS (sA_knee) and src-knee HTML.

| Exercise | Isometric modifier text |
|---|---|
| Slant Board Step Down | `Isometric modifier (soreness >2/10): lower to working angle, hold 30–45s each side instead of full eccentric reps.` |
| Split Squat Progression | `Isometric modifier (soreness >2/10): hold at bottom position 30–45s each side instead of reps.` |
| Curtsy Squat | `Isometric modifier (soreness >2/10): hold in crossed position at working depth 20–30s each side instead of reps.` |
| Cossack Squat | `Isometric modifier (soreness >2/10): hold at working depth 20–30s each side instead of reps.` |
| Hamstring Slide | `Isometric modifier (soreness >2/10): skip — substitute Hamstring Isometric in Flexion from the Optional pool.` |
