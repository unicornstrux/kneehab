Changes v11.1 — Day R: adaptive recovery & yoga
  • Day R — Recovery & Mobility added. Adaptive yoga/mobility session. Pick 2–3 categories based on what feels tight (Hips, Hamstrings, Thoracic, Quads/IT Band, Shoulders, or General Flow). Hold durations doubled from Day A/B (60s vs 30s) per Mitchell's dose-response model. No collagen, no knee eccentric loading. Insert anywhere in A/B rotation.
  • New exercises: Supine Spinal Twist (thoracic rotation gap fill), Thread-the-Needle, Standing Forward Fold, Child's Pose with Reach, Down Dog holds.
  • Lateral Quad Roll gets a proper home in Quads/IT Band category with extended duration (2 min each side).
  • General Flow option: ~12 min continuous sequence (Cat-Cow → Down Dog → Lizard → Pigeon → Twist → Child's Pose) for days with no specific complaint.
  • Day A time estimate corrected from ~45 min to ~55 min.
Changes v11.0 — Rehab → Training: A/B split, upper body mandatory, daily habits integrated
  • Program restructured from rehab to training. 4 day types (Load, Recovery, Activity, Daily Habits) → 3 day types (Day A Lower Body, Day B Upper Body, Activity). Alternating A→B→A→B with Activity days inserted as needed.
  • Upper body mandatory. Full upper body block moved from optional (Recovery day) to mandatory (Day B). Collagen required on Day B — upper body mechanical loading qualifies under Baar protocol.
  • Daily habits integrated. Tibialis Raise and Dead Hang moved into Primer block on both days. Short Foot, SL Balance, Backward Walk, Wall Slide remain as informal daily habits.
  • Primer block added. Tibialis Raise + Dead Hang at T+5 on both days — eliminates separate Daily Habits obligation for these exercises.
  • Mobility updated. World's Greatest Stretch → Lizard Pose (deeper sustained hip flexor/adductor stretch). Supine Hip ER with Block → Pigeon Pose (same deep rotator target, more functional position). Shoulder CARs added to mobility. All mobility exercises daily on both days.
  • Bird Dog now has 3 variations. Slow Return (eccentric anti-rotation), Elbow-to-Knee (dynamic), Bird Dog Row (loaded). Pick one per session. Provides clear progression path.
  • Hip Flexor exercise mandatory in Core block. Pick one per session: Hanging Knee Raise (highest demand), Tuck L-Sit, or Foot-Supported L-Sit. Addresses anterior pelvic tilt under fatigue.
  • Cossack Squat replaced by Lateral Step-Up. Cossack too demanding at current level. Lateral Step-Up provides frontal plane quad + glute loading with scalable progression (box height, tempo, load). Curtsy Squat and Lateral Step-Up alternate each session.
  • Split Squat and Curtsy Squat now include 2s pause at bottom. Standard prescription, not soreness modifier. Increases tendon adaptation and motor control at sticking point.
  • Soleus Calf Raise promoted to main Knee block. Primary plantarflexor under all knee-flexed positions.
  • Knee maintenance on Day B. SL Wall Sit (3×20s), Standing Ham ISO (2×20s), Copenhagen (2×20s) — isometrics only, reduced volume, no recovery conflict.
  • WLYT moved to Day B only. Pairs with upper body pulling. Day A runs Bird Dog only for spinal work.
  • Hollow Body / RKC Plank alternate on Day B. Both on every session was redundant. Day A keeps Hollow Body always.
  • Zone 2 cycling unrestricted. Phase 1 restriction removed. Collagen 45 min before any Zone 2 ride. Bellemans Bike remains a separate cartilage protocol — a Zone 2 ride does not replace it.
  • Short Ruck added. 20–30 min loaded pack, evening option on Day A or B. Not on the same day as a Zone 2 bike ride. Progression via 2×2 rule (duration → weight → terrain). No collagen needed — nocturnal GH pulse covers.
Changes from v7.0 — Floor L-Sit added
  • Floor L-Sit added to optional casual micro-session pool. Seated, heels on floor, hands beside hips, press down through straight arms. 2–3 × 20–30s. Unique contributions not covered elsewhere in the program: active shoulder depression under vertical load (lower trap, serratus anterior in compression pattern vs. shear pattern of Scap Push-Up) and tricep isometric. Partial overlap with hollow body hold (anterior core isometric) and RKC plank (co-contraction) but not redundant — different arm pattern and hip position. Not added to Core block — micro-session placement preferred because isometric adaptation is driven by frequency across the day, not single-session volume. Floor access required. No recovery demand. Progress by reducing heel support height over weeks.
Changes from v6.9 — Scientific review, protocol rebuild, progressive overload
  • Collagen protocol rebuilt. Day-label rules replaced with a single universal rule: 15g + 200mg vit C, 45–60 min before any qualifying mechanical load session, minimum 2h after last protein meal. Explicit load session table added. "No collagen on Day C" rule replaced with loading-event logic — collagen warranted on Day C only if upper body block is included.   • Lateral Side Step removed. Replaced by Lateral Step-Up (Day A) and Skater Squat (Day B).
  • Lateral Step-Up added — Day A. 15cm box, 3s eccentric, bodyweight Phase 1. Levers: 15cm→20cm→25cm / eccentric tempo / bodyweight→loaded.
  • Skater Squat added — Day B. Start assisted, partial depth. Levers: assisted→unassisted / partial→full depth / standard→3s eccentric / bodyweight→loaded.

Supine Hip ER = end-range isometric for deep external rotators — diagnosis-specific for PFPS, not replaceable by 90/90 hold.
  • 2×2 Rule added as universal progression gate. ≤2/10 pain during and morning after, two consecutive sessions → pull next lever. Pain >2/10 → hold one full week. Replaces hard phase transitions.
  • Progression levers added to all key exercises. Wall Sit, Step Down, Lateral Step-Up, Skater Squat, Split Squat, Calf Raise, Hamstring Slide, Short Arc Quad, Cossack Squat — each has a defined lever sequence. Program never hard-resets.

10cm step Phase 1. Lower opposite foot over 5s, 2s pause at bottom (foot hovering, not touching), return. Hand on wall → unassisted. Rear foot crosses behind and outside stance foot. Stance-leg glute med resists femoral adduction — the primary training stimulus. 2s pause at bottom for tendon adaptation. Keep stance-leg knee over 2nd–3rd toe. Apply asymmetry rule. Alternates with Lateral Step-Up each session. Regression: hold door frame, reduce depth. Stand beside box. Step up laterally, control 3s descent on the way down. Frontal plane quad + glute under single-leg load. Stand facing wall, slight knee bend (10–15° — barely perceptible). Press heel back and into floor as if dragging it toward you. Hold 20–30s.
